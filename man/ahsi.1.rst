==================
ahsi
==================

-----------------------------------------------------------
Hello world program using Crash Javascript
-----------------------------------------------------------
:Version: ahsi |version|
:Manual section: 1

Synopsis
========

ahsi


Description
===========

Hello world module written to show how to use
the Crash JavaScript library to write a program
which runs both in the browser as well as in
a node.js enviroment.
This is significant because the Crash JavaScript
library includes file writing functions.
